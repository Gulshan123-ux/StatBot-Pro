# StatBot Pro

StatBot Pro is organized as two active apps plus an archive for older scratch work.

## Structure

```text
StatBot-Pro-Atharva/
├── backend/
│   ├── app/
│   │   ├── models/
│   │   ├── routers/
│   │   ├── services/
│   │   └── utils/
│   ├── static/
│   ├── main.py
│   ├── requirements.txt
│   └── README.md
├── frontend/
│   ├── app/
│   ├── components/
│   ├── lib/
│   ├── public/
│   ├── types/
│   └── README.md
└── archive/
    └── legacy-prototypes/
```

## Notes

- The active backend lives only in `backend/`.
- The active frontend lives only in `frontend/`.
- Older scratch files and UI mockups are preserved under `archive/legacy-prototypes/`.
